# Meridian — Demonstração

Pequeno repositório de demonstração contendo o ficheiro `index.html` com uma UI de referência salarial.

Como usar:

- Abra `index.html` no navegador: clique duas vezes ou arraste para uma janela do browser.

Publicar no GitHub:

1. Crie um repositório remoto no GitHub (https://github.com/new).
2. Adicione o remote e envie os commits locais:

```powershell
Set-Location "C:\Users\celto\Downloads\salo"
git remote add origin https://github.com/USERNAME/REPO.git
git branch -M main
git push -u origin main
```

Substitua `https://github.com/USERNAME/REPO.git` pela URL do seu repositório.
